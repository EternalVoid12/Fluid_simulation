# Discription
Simple fluid simulation using Raylib library written in C/C++.
Based on this paper https://www.dgp.toronto.edu/public_user/stam/reality/Research/pdf/GDC03.pdf

# Usage
| keys          | Description      |
| ------------- | -----------------|
| SPACE         | Pause            |
| Left mouse    | Add density      |
| Right mouse   | Add velocity     |
| R             | Add densityx10   |
| F             | show u "velocity"|
| V             | show v "velocity"|
| A             | clear screen     |
